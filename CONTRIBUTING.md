# Contributing to Kaman AI Chat

Thank you for your interest in contributing to **Kaman AI Chat**! This project is open to contributions from the community. Whether you're reporting bugs, suggesting features, or submitting code, your help is appreciated.

## 🤝 How to Contribute

### 1. Reporting Bugs
If you encounter a bug, please open an [Issue](https://github.com/Mohammad-Hasan-Kaman/kaman-ai-chat/issues) and include:
- A clear title and description.
- Steps to reproduce the problem.
- Expected vs. actual behavior.
- Screenshots (if applicable).
- Your environment (OS, .NET version, API key setup).

### 2. Suggesting Features
Have an idea for a new feature? Please open an [Issue](https://github.com/Mohammad-Hasan-Kaman/kaman-ai-chat/issues) labeled `enhancement` and describe:
- The problem you're trying to solve.
- Your proposed solution.
- Why this feature is useful.

### 3. Submitting Code Changes
If you'd like to submit code changes:
1. **Fork** the repository.
2. Create a **new branch** from `main`:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes and ensure the code builds and runs.
4. Write **clear commit messages**:
   ```bash
   git commit -m "feat: Added voice response for AI"
   ```
5. **Push** your branch:
   ```bash
   git push origin feature/your-feature-name
   ```
6. Open a **Pull Request (PR)** against `main`.

### 4. Code Style
- Follow C# coding conventions.
- Keep methods and classes concise and readable.
- Add comments for complex logic.
- Ensure your code works with the existing API integrations.
